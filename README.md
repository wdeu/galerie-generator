# Bücherkiste – Booklooker Galerie

Automatischer Galerie-Generator für Booklooker-Verkäufer.

## Setup
```bash
pip3 install requests
cp ~/.booklooker-sync.ini.example ~/.booklooker-sync.ini
nano ~/.booklooker-sync.ini  # API Key eintragen
```

## Nutzung
```bash
./galerie-generator.py
# Dann: public/index.html auf IONOS hochladen
```
